Passionné par le développement web, je suis actuellement en transition vers ce domaine et sur le point de débuter une formation en Développement Web et Web Mobile (DWWM). Mon objectif est d’acquérir les compétences techniques nécessaires pour créer des sites web modernes et performants. Je suis particulièrement attiré par [exemple : le développement front-end, l’UI/UX design, la création d’applications web dynamiques, etc.]. Ce portfolio est mon premier projet personnel, conçu pour mettre en pratique mes premières connaissances et documenter mon évolution.
